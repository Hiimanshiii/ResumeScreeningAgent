# Resume Screening Agent

AI-powered resume screening and candidate ranking system built using:

- ASP.NET Core Minimal API
- Razor Pages
- Semantic Kernel
- Google Gemini
- SQL Server Express
- Entity Framework Core
- QuestPDF

## Features

- Resume Upload
- Job Description Upload
- AI Candidate Analysis
- Candidate Ranking
- Interview Question Generation
- PDF Report Generation
- SQL Server Persistence

## Tech Stack

ASP.NET Core + Semantic Kernel + Gemini + SQL Server + EF Core
